#Steps to ensure before running the script
1) In the "Set Variables - 1" section, update the following 2 variables to what you have set up in your own environment. 
	ec2_key_name
	ec2_security_group

2) All other variables can be changed if you wish, but will work fine if not

3) Make sure your key pair (".pem" file) is in the same directory as this script
